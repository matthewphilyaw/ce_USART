static SerialResult_t GetByte ( uint8_t *  destination)
static

Retrieve byte from interface.

Parameters
[out]destinationis the pointer to put the byte into
Returns
SERIAL_SUCCESS -> Byte successfully read without errors
SERIAL_CLOSE -> Interface is closed and nothing is done
SERIAL_INVALID_PARAMETER -> destination is a null pointer
SERIAL_NO_DATA -> No data to read from interface
SERIAL_OVER_RUN -> Byte in data register is valid, but next byte to be shifted in is being overwritten (I.E not reading fast enough)
SERIAL_FRAMING_ERROR -> Byte in data register is likely junk and issue with buadrate or other parameters thus interface is no longer in sync
SERIAL_PARITY_ERROR -> Byte in data register is likely junk and parity check failed
SERIAL_NOISE_ERROR -> Byte is data register is likely junk and noise was detected on the line
